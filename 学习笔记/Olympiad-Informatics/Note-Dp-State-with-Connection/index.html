基于连通性的状态压缩

容易发现，状态压缩是一个针对一维状态进行扩展的算法，而我们使用状压 时却往往解决的是二维网格图类型的问题，在对列进行较大限制时会造成空间与时间上的冗余，从而使得有些题目在没有其它方法的时候并没有办法通过维护基本状态通过。
